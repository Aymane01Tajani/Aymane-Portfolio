GymTrace - Application pour le Suivi personnel des Activités Sportives

Projet Personnel

Décembre 2024 Projet Personnel Application Mobile

Aperçu du projet

GymTrace est une application mobile de fitness développée avec Flutter, conçue pour aider les utilisateurs à suivre leurs entraînements et progrès de manière efficace. L'application permet aux utilisateurs de sauvegarder leurs séances d'entraînement, de suivre leur progression au fil du temps, et de consulter l'historique détaillé de leurs exercices. Avec une interface intuitive et moderne, GymTrace offre une expérience personnalisée pour optimiser les performances sportives et maintenir la motivation des utilisateurs dans leur parcours fitness.

Architecture technique

L'architecture de GymTrace est basée sur Flutter avec Firebase pour une expérience mobile native et une gestion cloud des données :

  • Frontend Flutter : Interface utilisateur développée avec Material Design pour une expérience native et fluide sur mobile.
  • Backend Firebase : Utilisation de l'écosystème Firebase pour l'authentification sécurisée et la gestion des services cloud.
  • Base de données Firestore : Cloud Firestore pour le stockage et la synchronisation en temps réel des données d'entraînement, profils utilisateur et historiques.
  • Authentification Firebase : Système de connexion sécurisé avec gestion des sessions utilisateur via Firebase Auth.
  • Navigation Flutter : Système de navigation optimisé avec AppBar personnalisée et routing entre les différentes pages.
  • Visualisation : Implémentation de graphiques et charts pour le suivi des progrès avec données synchronisées depuis Firestore.
