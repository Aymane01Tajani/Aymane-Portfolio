Une révolution dans la gestion de caisse arrive...

FlowPay est en cours de développement intensif. Cette application révolutionnaire transformera la façon dont vous gérez votre caisse, vos paiements et vos transactions commerciales. Restez connecté pour découvrir une solution complète qui simplifiera votre activité.

Aperçu du projet

FlowPay sera une application de caisse moderne et intuitive, développée avec les dernières technologies (Java, Spring Boot, Angular, Firebase et Cloudinary). Elle offrira une gestion complète des transactions, un système de paiement sécurisé, et des analyses en temps réel pour optimiser votre activité commerciale.

Technologies utilisées

Java Spring Boot Angular Firebase Cloudinary REST API
