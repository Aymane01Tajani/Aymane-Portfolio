Online Shop - APPLICATION WEB E-COMMERCE

Projet Personnel

Novembre 2024 Projet Personnel Application WEB

Aperçu du projet

OnlineShop est une plateforme web e-commerce moderne et intuitive développée en C# avec ASP.NET MVC, reposant sur une architecture Entity Framework Core en Code First. Conçu pour offrir une expérience fluide aux utilisateurs, le site intègre un catalogue complet de produits, une page de détails permettant de passer commande directement via WhatsApp, ainsi qu’un espace administrateur sécurisé offrant la gestion complète des articles grâce aux opérations CRUD. Alliant performance, simplicité d’utilisation et automatisation des tâches, OnlineShop constitue une solution efficace pour digitaliser les ventes et optimiser la gestion commerciale au quotidien.

Architecture technique

Architecture technique de OnlineShop

L'architecture de OnlineShop repose sur une approche moderne et scalable :

  • Frontend ASP.NET MVC : Interface utilisateur responsive, des vues développées avec Bootstrap, HTML et CSS pour une expérience optimale.
  • Backend C# : Logique métier robuste avec contrôleurs MVC et services.
  • Entity Framework Core : ORM Code First pour la gestion de base de données.
  • SQL Server : Base de données relationnelle pour la persistance des données.
  • WhatsApp API : Intégration pour les commandes directes via WhatsApp.
  • Admin Dashboard : Interface d'administration pour la gestion complète des produits.
