Aperçu du rôle

En tant que Consultant Technico-Fonctionnel Odoo chez Blackswan Technology, je joue un rôle central en tant qu'interface entre les équipes métier et les équipes techniques. Mon objectif principal est de transformer les besoins complexes des clients en solutions Odoo optimisées et performantes, assurant une mise en œuvre réussie et une adoption élevée par les utilisateurs finaux.

Technologies et outils utilisés

Odoo ERP Python XML PostgreSQL JavaScript Git GitHub Visual Studio Code Linux

Responsabilités principales

  • Analyse et compréhension des besoins métier des clients, documentation des processus existants et identification des opportunités d'optimisation
  • Design de solutions Odoo personnalisées adaptées aux spécificités de chaque client, en minimisant les développements custom
  • Configuration des modules Odoo (Finance, Ventes, Achats, Stock, Ressources Humaines, etc.) selon les requirements métier
  • Développement de modules et extensions personnalisés en Python pour répondre à des besoins spécifiques
  • Gestion du cycle complet d'implémentation : planification, exécution, tests et rollout
  • Formation des utilisateurs finaux et création de documentation technique
  • Support post-déploiement, optimisation des performances et amélioration continue du système
  • Collaboration avec les équipes techniques pour l'intégration Odoo avec d'autres systèmes (APIs, web services)
